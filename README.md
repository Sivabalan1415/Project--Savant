# Project--Savant
SAVANT - Landing Page Website

Overview:
SAVANT is a responsive landing page designed for promoting a mobile application.

Features:
- Responsive design using Bootstrap
- Navigation bar with tooltips
- Modal popups (Join Now / Free Trial)
- Team section with social media popovers
- Testimonials and FAQ section
- Contact form and newsletter subscription

Technologies Used:
- HTML5
- CSS3 / SCSS
- Bootstrap 3
- Font Awesome
- JavaScript

Project Structure:
SAVANT/
│── index.html
│── css/
│   ├── bootstrap.min.css
│   ├── font-awesome.css
│   ├── style.css
│   ├── styleRW.css
│── js/
│   ├── bootstrap.min.js
│── images/
│── fonts/

Setup Instructions:
1. Download or clone the project
2. Open index.html in any browser

Functionalities:
- Smooth navigation between sections
- Tooltip and popover interactions
- Modal popup functionality
- Fully responsive layout

Notes:
- SCSS variables used for styling
- Ensure correct file paths for assets
